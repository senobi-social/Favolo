# Favolo
・Twitterのいいね！をテーマに沿って紹介できるアプリ

・いいね！をしてデザインを選ぶだけでオリジナルな発信ができる

・企業のコンテンツ紹介なども可能

### 機能面
・TwitterAPIの取得機能（APIトークンを使用）

・フォロー機能（表示まで）

・いいね機能（カウント表示ができる）

・タグ機能（１つまで）

・設定機能（パスワード、コメント、デザイン、タグの変更ができる）

### 問題点 
・素のSQL文を記述している

・モデルを利用していない

・関数ベースビューの記述になっている

・アプリケーションの分割ができていない

・コードが冗長

・変数の取り扱いが煩雑

### β版への移行

機能の拡充、可読性の向上、Djangoの最大利用のためにもβ版に移動する
